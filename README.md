# DbD Evolution: Resurrection

「Dead by Daylight」をモチーフにした、インフレ系クリック・放置ゲームです。

## ゲーム概要
修理ゲージを溜めて発電機を完成させ、獲得したBP（ブラッドポイント）やIS（イリデセントシャード）を使って自身を強化していきます。
さらに「昇華（Ascension）」や「超越（Transcendence）」を繰り返すことで、異次元のインフレを体験できます。

## 主な特徴
- **多彩なスキン**: 8種類の視覚的なテーマを切り替え可能。
- **重層的な成長システム**:
  - **BPアップグレード**: 基本的な修理速度や判定の強化。
  - **ISアップグレード**: 自動修理やIS獲得効率の向上。
  - **昇華（AP）**: 強力なAPバフとスキンの解放。
  - **超越（TP）**: 圧倒的な倍率と自動昇華などの究極バフ。
- **スキルチェック**: DbDおなじみのスキルチェック。Great判定で爆発的な進行が可能。
- **インフレ**: 倍率が指数関数的に増大する快感。

## 遊び方
1. 画面をマウスで **HOLD（長押し）** して修理を開始します。
2. 判定音が鳴ったら、針がゾーン重なる瞬間に **RELEASE（離して）** スキルチェックを行います。
3. 獲得したBPでアップグレードを購入します。
4. 詰まってきたら「昇華」や「超越」を行って、さらに強力な力を手に入れましょう。
